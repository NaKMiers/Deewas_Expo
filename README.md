# Deewas

Deewas is a modern, user-friendly expense management app designed to help you track and manage your personal finances effortlessly. With features like multi-wallet support, insightful charts, budget planning, and a smart AI assistant, Deewas empowers you to take control of your financial life.

## Features

- **Clean Interface:** Intuitive navigation and management.
- **Multi-Wallet Support:** Organize your finances across multiple wallets.
- **Insightful Charts:** Visualize your spending with interactive charts.
- **Budget Planning:** Set budgets and achieve your financial goals.
- **Smart AI Assistant:** Get help managing transactions with AI.
- **Custom Categories:** Classify transactions for better tracking.
- **Secure & Private:** Your data stays on your device.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/) or [npm](https://www.npmjs.com/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/your-username/deewas-expo.git
   cd deewas-expo
   ```

2. Install dependencies:

   ```sh
   yarn install
   # or
   npm install
   ```

3. Copy and configure environment variables (if needed):

   ```sh
   cp .env.example .env
   # Edit .env as needed
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```
